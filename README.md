# Tesla Motors (tesla-motors)

Tesla, Inc. (formerly Tesla Motors, Inc.) is an American electric vehicle and clean energy company that designs and manufactures electric cars, battery energy storage systems, solar panels, and related products. The Tesla Owner API provides programmatic access to Tesla vehicles for monitoring state, controlling climate, locking/unlocking doors, managing charging, and executing remote commands.

## APIs

### Tesla Owner API

The Tesla Owner API (unofficial) provides access to Tesla vehicle telemetry, state data, and remote command capabilities for personal vehicles. Supports vehicle listing, state retrieval (charge, climate, drive, vehicle config), and commands including wake-up, door lock/unlock, climate control, charging, trunk control, and media playback.

- **Human URL:** [https://tesla-api.timdorr.com/](https://tesla-api.timdorr.com/)
